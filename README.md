# 🎓 대학 진로 상담 시스템

대학교 및 학과 정보를 제공하고, 적성검사를 통해 자신에게 맞는 진로를 찾을 수 있는 AI 기반 상담 시스템입니다.

## 📌 주요 기능

### 1. 대학 정보 챗봇 💬
- 전국 주요 대학 정보 제공
- 학과별 상세 정보 (취업률, 평균연봉, 필요역량 등)
- 대학 진학률 통계
- AI 챗봇을 통한 실시간 상담
- 표와 그래프를 통한 시각화

### 2. 적성검사 ✨
- 5개의 질문으로 간단한 적성 검사
- 성향 분석 (탐구형, 예술형, 사회형, 현실형)
- 적성에 맞는 학과 추천
- 추천 학과의 상세 정보 제공

### 3. 사이드바 기능 📊
- **주로 검색하는 것**: 자주 검색한 키워드 순위
- **예전 대화 내용**: 최근 대화 기록 요약

## 🚀 시작하기

### 필요 조건
- Python 3.8 이상
- pip (Python 패키지 관리자)

### 설치 방법

1. 저장소 클론 또는 다운로드
```bash
cd "D:\20313 지현우"
```

2. 필요한 패키지 설치
```bash

```

3. 애플리케이션 실행

## 💡 사용 방법

### 1. 홈 화면
- **대학 정보 보기**: 대학 및 학과 정보를 챗봇에게 질문
- **적성검사 시작**: 5개 질문에 답하고 맞춤 진로 추천 받기

### 2. 대학 정보 챗봇 사용
1. 질문 입력창에 궁금한 내용 입력
2. 챗봇이 관련 정보 제공
3. "표나 그래프를 보여줄까요?" 질문에 응답
4. ✅ 또는 ❌ 버튼으로 시각화 선택

**예시 질문:**
- "서울대학교에 대해 알려주세요"
- "컴퓨터공학과 취업률은 어떻게 되나요?"
- "최근 대학 진학률은 어떤가요?"
- "의예과 정보를 알려주세요"

### 3. 적성검사
1. 5개의 질문에 차례대로 답변
2. 진행 상황을 진행 바로 확인
3. 완료 후 성향 분석 결과 확인
4. 추천 학과 및 상세 정보 확인
5. 필요시 다시 검사하거나 대학 정보로 이동

### 4. 사이드바 활용
- **주로 검색하는 것**: 인기 검색어 순위 확인
- **예전 대화 내용**: 이전 대화 내용 다시 보기
- **홈으로**: 언제든지 메인 화면으로 돌아가기

## 📊 데이터 구조

### 대학 정보 (university_info.csv)
- 대학명, 위치, 설립연도, 학생수
- 주요학과, 평균등급, 취업률

### 학과 정보 (major_info.csv)
- 학과명, 분야, 평균연봉, 취업률
- 필요역량, 추천적성

### 진학률 정보 (admission_rate.csv)
- 연도별 대학진학률, 4년제/전문대 진학률
- 재수생 비율
